# Homebrew Satori

This is a Homebrew tap for [Satori CLI](https://github.com/KunalKatariya/satori-cli) - a real-time multilingual transcription and translation tool with GPU acceleration.

## Installation

```bash
brew install KunalKatariya/satori-cli/satori-cli
```

## Usage

After installation, run:

```bash
satori init          # Interactive setup
satori devices       # List audio devices
satori translate     # Start transcribing
```

For more information, visit the [main repository](https://github.com/KunalKatariya/satori-cli).
